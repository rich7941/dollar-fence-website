# Dollar Fence Website

Professional fence company website - Dollar Fence

## About This Repository

This repository contains the complete Dollar Fence website files, providing quality fence installation and repair services for residential and commercial properties across the USA.

## Contents:
- All HTML pages (homepage, about, services, contact, etc.)
- All location pages for Texas and Florida cities
- CSS stylesheets and JavaScript files
- Sitemap and documentation

## Deployment:
These files can be deployed to any static web hosting service:
- Upload all files to your web server's document root
- Ensure index.html is set as the default page
- No server-side processing required

## Structure:
- index.html - Homepage
- about/ - About page
- services/ - Services page
- contact/ - Contact page
- fence-types/ - Fence types page
- locations/ - Location pages
- assets/ - CSS and JavaScript files
- sitemap.xml - SEO sitemap
